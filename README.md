Data Extraction pipeline and NCCN Guidlines based RAG Chatbot.

Retrieval-Augmented Generation (RAG) chatbot designed to assist clinicians in accessing guideline-based information quickly and accurately. The system combines automated document preprocessing, extraction of both textual and visual information, semantic retrieval using vector databases, and controlled language generation using large language models (LLMs)
